## 📋 Инструкция по запуску после мерджа:

### 1. Клонировать репозиторий (если нужно):
```bash
git clone https://github.com/doboleshev/restaurant_booking.git
cd restaurant_booking
```

### 2. Создать виртуальное окружение:
```bash
python -m venv venv
venv\Scripts\activate  # Windows
# или
source venv/bin/activate  # Linux/Mac
```

### 3. Установить зависимости:
```bash
pip install -r requirements.txt
```

### 4. Применить миграции:
```bash
python manage.py migrate
```

### 5. Создать суперпользователя (опционально):
```bash
python manage.py createsuperuser
```

### 6. Создать тестовые столики (через админ-панель или скрипт):
- Зайти в админ-панель: http://127.0.0.1:8000/admin/
- Добавить столики вручную или использовать скрипт

### 7. Запустить сервер:
```bash
python manage.py runserver
```

### 8. Открыть в браузере:
- Сайт: http://127.0.0.1:8000/
- Админка: http://127.0.0.1:8000/admin/
