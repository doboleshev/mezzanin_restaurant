# Ресторан "Мезонин" - Система бронирования столиков

Полнофункциональное веб-приложение для бронирования столиков в ресторане, разработанное на Django с использованием Bootstrap 5.

## 🚀 Основные функции

- **Онлайн бронирование** столиков с выбором даты и времени
- **Личный кабинет** для управления бронированиями
- **Авторизация и регистрация** пользователей
- **Административная панель** для управления контентом
- **Адаптивный дизайн** для всех устройств
- **Форма обратной связи**

## 🛠 Технологический стек

- **Backend**: Django 6.0 (Python)
- **Frontend**: HTML5, CSS3, JavaScript, Bootstrap 5
- **Database**: SQLite3 (для разработки)
- **Templates**: Django Template Language
- **Icons**: Bootstrap Icons
- **Styling**: Custom CSS with Bootstrap 5

## 📦 Установка и настройка

### Создание виртуального окружения
```bash
python -m venv venv
```

### Активация окружения

**Windows:**
```bash
venv\Scripts\activate
```

**Linux/Mac:**
```bash
source venv/bin/activate
```

### Установка зависимостей
```bash
pip install -r requirements.txt
```

### Настройка базы данных

**Применение миграций:**
```bash
python manage.py migrate
```

**Создание администратора:**
```bash
python manage.py createsuperuser
```

### Запуск сервера разработки
```bash
python manage.py runserver
```

## 🌐 Доступ к приложению

- **Сайт**: http://127.0.0.1:8000/
- **Админка**: http://127.0.0.1:8000/admin/

## 🎯 Особенности реализации

### Пользовательская часть
- Интуитивная форма бронирования с валидацией
- Календарь доступности столиков
- История бронирований в личном кабинете
- Уведомления о статусе бронирования

### Административная часть
- Управление столиками (добавление, редактирование, удаление)
- Просмотр и подтверждение бронирований
- Управление пользователями
- Обратная связь от клиентов

### Дизайн и UX
- Полностью адаптивный дизайн
- Современный интерфейс с Bootstrap 5
- Интерактивные элементы с JavaScript
- Плавные анимации и переходы
